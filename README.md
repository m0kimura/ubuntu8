UBUNTU 18.04LTS 基底版
====

UBUNTU 18.04 LTSベースでさまざまなドッカーモジュールの起点となるDockerモジュール

## 概要

  1. 環境変数として日本語環境を設定
  2. dockerユーザーとしての基本環境を作成
  3. 日本語環境としてフォントインストール、ロケール設定
  4. software-properties-common インストール
  5. wget curl nanoインストール
  6. gitインストール
  7. nodeは8.11.2をインストール
